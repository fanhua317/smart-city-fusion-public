# 第十轮按图选择实验记录

生成时间：2026-06-06

## 目标

前九轮证明单一全局参数很难同时拉高 `AG/SF/Qabf` 并保持 `SSIM/Nabf/CC/SCD`。第十轮改为按图选择：从已有候选池中为每一张测试图选择更合适的融合结果，再把 93 张组合成一个新的提交版本。

## 方法

- 新增脚本：`scripts/generate_round10_imagewise_selector.py`
- 候选池：从 `results/candidates*` 中读取完整候选，并保留预测前列和人工指定的极端样本。
- 选择策略：
  - 预计算每张图、每个候选的本地 13 项指标。
  - 用 rank-points 初始化，分别偏向 balanced/detail/safe_qabf。
  - 用贪心逐图替换，直接优化校准后的插榜平均名次。
- 输出目录：`results/candidates_round10/`
- 分数表：`results/round10_local_scores.csv`
- 候选池表：`results/round10_pool.csv`

## 判断

第十轮是目前本地预测最有价值的一轮：

- `03_rankpoints_detail_rank` 的预测插榜平均名次约 `19.62`，优于第九轮和第四轮微边缘候选。
- `05_rankpoints_safe_qabf_rank` 的预测 Qabf 更高、Nabf 更低，适合作为第二上传对照。
- 风险是按图选择会让不同图像来自不同算法，若官方指标包含未复刻的跨图一致性惩罚，可能有偏差；但当前官方 13 项平均指标更适合这类组合策略。
